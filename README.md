# 🌱 Irrigation Prediction — End-to-End ML Project

A full-stack machine learning project built as a learning exercise, covering the entire pipeline from raw data to a deployed prediction API with a React frontend.

This model was built for a Kaggle competition on irrigation need prediction, evaluated on balanced accuracy between predicted and observed classes.
---

## What This Project Covers

This project was built to learn and practice the following skills in one cohesive real-world scenario:

- **Data preprocessing** with custom sklearn transformers
- **Model training** with cross-validation and hyperparameter tuning
- **Model deployment** with FastAPI
- **Frontend development** with React
- **Full-stack integration** between a Python backend and a JavaScript frontend
